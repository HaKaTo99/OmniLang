# Getting Started OmniLang v0.1

## Instalasi
- Pastikan Python 3.x terinstal di sistem Anda.
- Clone atau download repository OmniLang.

## Menjalankan Interpreter
- Jalankan perintah berikut di terminal:
  ```
  python src/omnilang.py examples/hello.omni
  ```
- Output yang diharapkan:
  ```
  Hello, OmniLang!
  ```

## Struktur File
- `src/omnilang.py` : Interpreter utama
- `examples/hello.omni` : Contoh program
- `tests/` : File pengujian
