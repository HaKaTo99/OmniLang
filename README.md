# OmniLang Compiler (omc)

Compiler "Military Grade" berperforma tinggi yang dikembangkan secara harmonis dan selaras dari Fase 1 hingga Fase 10.

## Fitur Utama
- **Stability First**: Menggunakan State Machine untuk Backend Rust, menjamin control flow yang aman tanpa `goto`.
- **Hoisting Progresif**: Menghindari masalah scope pada Rust melalui hoisting variabel otomatis.
- **Optimasi Cerdas**: Constant folding untuk mengurangi overhead runtime.
- **Built-in Runtime**: Dukungan fungsi `print` dan tipe data dasar.

## Cara Penggunaan
1. Jalankan compiler:
   ```powershell
   cd omc
   cargo run
   ```
2. Kompilasi hasil output:
   ```powershell
   rustc output.rs
   ```
3. Jalankan binary:
   ```powershell
   .\output.exe
   ```
